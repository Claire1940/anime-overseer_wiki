<Section icon={Sparkles} title="リセマラ用リソースのアクティブコード">

**Anime Overseer リセマラガイド**の戦略を成功させるには、大量のリソースが必要です。開発者は、リセマラに不可欠な**特性ディスク（Trait Discs）**、**ソウルクリスタル（Soul Crystals）**、**ジェム（Gems）**を入手できるコードを頻繁に配布しています。

| コード名 | 主な報酬 | 最適な使い道 |
| :--- | :--- | :--- |
| **rerollcode** | 特性ディスク 75個, スーパーソウルクリスタル 75個 | Sランクユニット用の大量特性リセマラ |
| **qolfixesforyouall!** | ジェム 2,500個, ゴールド 2,500, 特性ディスク 25個 | 初期召喚および基本的なアップグレード |
| **wunbosavedtheday!** | 特性リセマラ 50回分 | キャリーユニットの特定特性狙い |
| **likethegame!** | ジェム 3,000個, 特性ディスク 50個, 肉 50個 | アカウント全体の強化およびユニットの育成 |
| **AXISTHEGOAT** | ディスクリセマラ 15回分, ソウルクリスタル 10個 | ピンポイントなリソース管理 |

<Callout type="warning" title="リソース管理">
コードには有効期限があります。ゲームの序盤で手動で集めるのが最も難しいリソースである**特性リセマラ（Trait Rerolls）**が手に入るコードの引き換えを、常に最優先してください。
</Callout>

これらのコードを使用することで、何時間もの周回（グラインド）をスキップできます。例えば、`rerollcode` を使用すれば、1体のユニットに対して複数回の特性変更を試みるのに十分な素材が手に入ります。これは、せっかく高レアリティのキャラクターを引いたのに特性がイマイチだった場合に不可欠です。

</Section>

<Section icon={Swords} title="特性リセマラの仕組みとユニットの優先順位">

リソースごとの違いを理解することが、この**Anime Overseer リセマラガイド**の要となります。ゲーム中では、**特性ディスク（Trait Discs）**、**ディスクリセマラ（Disc Rerolls）**、**特性リセマラ（Trait Rerolls）**が登場します。それぞれ進行において異なる役割を持っています。

<CardGrid cols={3}>
  <Card title="特性ディスク (Trait Discs)">
    - **標準リソース**: 一般的な特性の進行に使用されます。
    - **入手量**: `500LIKES` などの節目記念コードで多く配布されます。
    - **用途**: BまたはAランクのユニットに「そこそこ良い」ステータスを付与するのに最適です。
  </Card>
  <Card title="特性リセマラ (Trait Rerolls)">
    - **プレミアムリソース**: ユニットの固有特性を直接変更します。
    - **希少性**: 入手困難なため、Sランクユニットのために温存してください。
    - **用途**: 高いダメージ倍率や射程補正を狙う際に使用します。
  </Card>
  <Card title="ソウルクリスタル (Soul Crystals)">
    - **進化素材**: ユニットの長期的な強さに直結します。
    - **重要ルート**: 進化の準備として、リセマラと並行して使用します。
    - **用途**: 恒久的に使い続ける予定のユニットのために保存します。
  </Card>
</CardGrid>

どのユニットから優先してリセマラ（特性変更）すべきかを決める際は、貴重な素材を一時的な繋ぎのユニットに浪費しないよう、以下の優先度表を参考にしてください。

| ユニットランク | リセマラ優先度 | 推奨される特性タイプ |
| :--- | :--- | :--- |
| **Sランクキャリー** | **最高** | ダメージ％、攻撃速度、全体射程 |
| **Aランクサポート** | **中** | クールダウン短縮、バフ持続時間 |
| **Bランク穴埋め** | **低** | 基本ステータス上昇（プレミアムリセマラは使用しない） |
| **進化対象** | **高** | 進化後の基礎ステータスに応じてスケールする特性 |

<Callout type="success" title="エキスパートのアドバイス">
「Bランク」のユニットであっても、優秀な「Sランク」の特性（広範囲AOEなど）を持っていれば、微妙な「Dランク」の特性（微量のHP自動回復など）を持つ「Sランク」ユニットを上回ることが多々あります。ユニットのスキル構成と特性のシナジーを常に意識しましょう。
</Callout>

<Section icon={Target} title="ステップ・バイ・ステップ リセマラ手順">

**Anime Overseer リセマラガイド**の手法を用いて、効率的に進行をリセットしたり、手持ちのユニットを最適化したりするには、以下の手順に従ってください。

<Steps>
  <Step num="1" title="初期アカウントのセットアップ">
    Robloxのページからゲームに参加します。有料アクセスを利用しない場合は、AFKチャンバーに入り、入場可能になるのを待ちます。メインハブに入ったら、**Codes（コード）**ボタンを探します。
  </Step>
  <Step num="2" title="リソースの全投入">
    ガイドに掲載されているすべてのアクティブコードを入力します。まずは**ジェム（Gems）**が手に入るコードを最優先してください。召喚（ガチャ）エリアへ向かい、手に入れたジェムをすべて使い切って初期メンバーを揃えます。
  </Step>
  <Step num="3" title="ランクの評価">
    引いたユニットを確認します。AランクまたはSランクのディーラー（アタッカー）が1体も出なかった場合は、アカウントの作り直し（リセマラ）を検討してください。優秀なキャリーを確保できている場合は、特性の管理に進みます。
  </Step>
  <Step num="4" title="特性の適用">
    手に入れた**特性ディスク（Trait Discs）**と**特性リセマラ（Trait Rerolls）**を最強のユニットに使用します。ダメージまたは射程が少なくとも15%以上アップする特性が出たら、一旦そこで止めましょう。最初から「完璧な」特性を追い求めすぎる必要はありません。
  </Step>
  <Step num="5" title="ウェーブでの実戦テスト">
    チームを編成してバトルに挑戦します。リセマラしたユニットの活躍度を確認しましょう。効率よくウェーブをクリアできているようであれば、残りの**ソウルクリスタル（Soul Crystals）**を進化のために貯め始めます。
  </Step>
</Steps>

<Callout type="info" title="リセマラのタイミング">
リセマラを行う最適なタイミングは、大型アップデートの直後や、「CCU（同時接続者数）」の節目（`THANKYOUFOR1KCCU` などのコードが配布されるタイミング）です。これらは通常、最も豪華なリソースが配布されるためです。
</Callout>

</Section>
