<Section icon={Sparkles} title="進化＆育成ガイド">

進化は、Anime Overseerにおいてユニットが獲得できる最も劇的なパワーアップです。基礎ステータスが上昇するだけでなく、新しいアビリティが解放されたり、攻撃範囲が広がったりするため、より強力な敵のウェーブに対処するためには不可欠です。ただし、進化はコストが高く、コード報酬や高レベルのファームでしか手に入らない特別な素材を必要とします。

<Steps>
  <Step num="1" title="コアチームの確立">
    進化を行う前に、実用的なユニットで編成を固めましょう。レベル1の暫定ユニットばかりのチームを、進化したユニット1体だけでキャリーすることはできません。
  </Step>
  <Step num="2" title="先にレベル上限まで強化">
    ゴールドやコインを使用して、ユニットを現在のレベル上限まで引き上げます。進化は、すでに活躍しているユニットに適用したときに最も高い効果を発揮します。
  </Step>
  <Step num="3" title="素材の収集">
    ソウルクリスタル、スーパーソウルクリスタル、そして肉（Meat）やオニギリ（Onigiri）などの特定の育成アイテムを集めましょう。これらの素材は、`rerollcode` や `THX4500` などの有効なコードから入手できます。
  </Step>
  <Step num="4" title="進化の実行">
    長期的なチーム計画に合致するユニットのみを進化させましょう。協力プレイや無限モードなどの進行に役立つユニットを優先します。
  </Step>
</Steps>

<Callout type="warning" title="リソースの希少性">
スーパーソウルクリスタルは非常に貴重です。Cティアのユニットや、1週間以内に使わなくなるようなキャラクターに消費しないでください。育成の壁にぶつからないよう、Sティアのキャリーユニットのために温存しておきましょう。
</Callout>

| 素材 | 主な入手方法 | 最適な使い道 |
| :--- | :--- | :--- |
| **ソウルクリスタル** | コード、ボスドロップ | 通常ユニットの進化。 |
| **スーパーソウルクリスタル** | 節目記念コード | 上位/レジェンダリーユニットの進化。 |
| **遺物の破片** | 無限モード | 高度な育成やレリックビルド。 |
| **食べ物（肉/オニギリ）** | コード、ログイン報酬 | ユニットの一般的な成長とレベルアップ。 |

進化させたユニットを実際の戦闘シナリオでテストすることが最終ステップです。進化前のパフォーマンスと比較して、クリア速度や生存率がどれだけ向上したかを確認しましょう。もし改善がわずかである場合は、配置戦略を調整するか、より良い特性（Traits）を探す必要があります。

</Section>

<Section icon={Shield} title="特性とリロール戦略">

特性（Traits）は、ユニットの性能を劇的に変化させるパッシブ効果として機能します。Anime Overseerでは、いくつかの有効なコードから特性ディスク（Trait Discs）やリロールアイテムを入手できるため、序盤から使いたくなるかもしれません。しかし、プロのプレイヤーは、永続的に使用するメタユニットを確保するまでこれらのリソースを蓄積しておきます。

<CardGrid cols={3}>
  <Card title="特性ディスク">
    一般的な特性の進行に使用します。新しいガチャに備えて、常に50個以上はストックしておきましょう。
  </Card>
  <Card title="特性リロール">
    微調整のための最も価値のあるリソースです。Sティアのキャリーユニットにのみ使用してください。
  </Card>
  <Card title="ディスクリロール">
    現在の特性スロットの品質を向上させるために特化したリソースです。
  </Card>
</CardGrid>

<Callout type="info" title="リロールのタイミング">
リロールを行う最適なタイミングは、ユニットを進化させた後です。これにより、キャラクターの最終形態に対して貴重なリソースを投資することになり、特性の長期的な価値を最大化できます。
</Callout>

| リソース | 推奨コード | 使用戦略 |
| :--- | :--- | :--- |
| **特性ディスク** | `likethegame!` | 長期的に使用する主力メンバーのために温存。 |
| **特性リロール** | `wunbosavedtheday!` | 価値の高いキャリーユニットにのみ使用。 |
| **ディスクリロール** | `AXISTHEGOAT` | ユニットの特性タイプが適していない場合に使用。 |

よくある間違いは、一時的な繋ぎのユニットに対して「完璧な特性」を追い求めてしまうことです。ユニットがBティア以下の場合、どのような特性を持たせていても、基礎ステータスやアビリティの限界により最終的には通用しなくなります。特性ディスクは、強化によってスケールしやすく、終盤のウェーブで明確な役割を持つユニットに集中させましょう。

</Section>

<Section icon={Map} title="効率的なジェムとゴールドの稼ぎ方">

Anime Overseerのメタユニットで強力な編成を維持するためには、ジェムとゴールドを安定して獲得する必要があります。ジェムは主に新しいユニットの召喚に使用され、ゴールドはユニットの強化やゲーム内の進行を支える命綱となります。

| リソース | 最適なファーム方法 | 消費優先度 | プレイヤーへのヒント |
| :--- | :--- | :--- | :--- |
| **ジェム** | コード、節目記念報酬 | 高 | 消費する前にバナーのラインナップを確認する。 |
| **ゴールド** | ウェーブクリア、デイリーミッション | 高 | 新ユニット獲得時のために一定の予備を残しておく。 |
| **ソウル** | リソースステージ | 中 | ウェーブを素早く処理できるユニットに集中する。 |

<Callout type="tip" title="コードの重ねがけ">
新しいコードは、アップデート時やゲームが「いいね」の目標数に達した際によく配布されます。パッチがあたった後は、必ず公式DiscordやRobloxのページをチェックして、無料のジェムとゴールドを手に入れましょう。
</Callout>

効率的なファームが鍵となります。ソロでプレイする場合は、失敗せずに安定して到達できる最高ウェーブ数に焦点を当ててください。協力プレイ（Co-op）モードでは、より奥のウェーブまで進めるため、時間あたりの報酬（遺物の破片やゴールドなど）が美味しくなる傾向があります。序盤のウェーブを素早くクリアしファームの効率を上げるために、コストパフォーマンスが最も高いユニットのアップグレードにゴールドを使いましょう。

</Section>
