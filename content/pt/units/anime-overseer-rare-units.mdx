<Section icon={Swords} title="Caminho de Evolução e Upgrade para Unidades Raras">

A evolução é a mecânica que transforma **unidades raras de anime overseer** padrão em forças imparáveis. Este processo requer materiais específicos como Soul Crystals, Super Soul Crystals e Relic Shards. O salto de poder de uma unidade base para sua forma evoluída é frequentemente de 200% ou mais, desbloqueando novas habilidades e aumentando significativamente o alcance.

<Steps>
  <Step num="1" title="Alcançar o Nível Máximo">
    Certifique-se de que sua unidade base atingiu o nível máximo através de experiência de batalha ou itens de comida como Carne e Onigiri.
  </Step>
  <Step num="2" title="Reunir Materiais">
    Colete os Soul Crystals e Relic Shards necessários. Muitos deles estão disponíveis através de códigos ativos como `rerollcode` e `THX4500`.
  </Step>
  <Step num="3" title="Evoluir no Hub">
    Visite o NPC de evolução no hub principal. Confirme se você tem Ouro suficiente para cobrir a taxa de transformação.
  </Step>
  <Step num="4" title="Rolar Traits Novamente">
    Uma vez evoluída, use seus Trait Rerolls para garantir uma trait de alto nível como "Divino" ou "Único" para complementar os novos status.
  </Step>
</Steps>

| Material | Como Farmar | Uso |
| :--- | :--- | :--- |
| **Soul Crystals** | Estágios de Chefes, Códigos | Evolução básica para unidades raras |
| **Super Soul Crystals** | Modo Infinito, Códigos | Evolução avançada para unidades S-tier |
| **Relic Shards** | Eventos Especiais, Marcos | Desbloquear passivas únicas de unidades |
| **Comida (Onigiri/Carne)** | Recompensas Diárias, Códigos | Subir o nível das unidades instantaneamente |

<Callout type="info" title="Dica de Evolução">
Só evolua unidades para as quais você tenha um plano de trait "Divino" ou "Soberano". A evolução fixa o potencial base da unidade, tornando a trait ainda mais impactante do que era na forma base.
</Callout>

</Section>

<Section icon={Shield} title="Trait Rerolls e Otimização de Recursos">

Traits são modificadores passivos que podem mudar drasticamente o desempenho de suas **unidades raras de anime overseer**. Como os Trait Rerolls e Trait Discs são limitados, você deve gastá-los estrategicamente. Um erro comum entre novos jogadores é usar todos os seus rerolls em unidades de nível A antes mesmo de terem invocado seu primeiro carry S-tier.

O jogo oferece várias maneiras de adquirir esses recursos por meio de códigos. Por exemplo, o código `wunbosavedtheday!` fornece 50 Trait Rerolls, o que é um impulso massivo para qualquer conta.

| Nome da Trait | Efeito | Melhor Para |
| :--- | :--- | :--- |
| **Divino** | +50% de Dano, +10% de Alcance | Todas as Unidades Carry |
| **Veloz** | +20% de Velocidade de Ataque | Unidades de Suporte/Atordoamento |
| **Atirador** | +30% de Alcance | Unidades AoE de longa distância |
| **Rico** | +15% de Ouro Ganho | Posicionamento em Ondas Iniciais |
