<Section icon={Sparkles} title="Guia de Progressão e Evolução de Unidades">

No Anime Overseer, apenas invocar uma unidade não é suficiente para sobreviver às hordas mais difíceis. Você deve melhorar ativamente e, eventualmente, evoluir seus combatentes para desbloquear o verdadeiro potencial deles. A evolução aumenta significativamente os atributos básicos de uma unidade e, muitas vezes, adiciona novas habilidades ou aumenta o alcance de ataque delas, o que é vital para concluir o conteúdo de fim de jogo.

<Steps>
  <Step num="1" title="Estabeleça uma Equipe Principal">
    Antes de investir pesado em evolução, monte uma equipe de 4 a 6 unidades que cubram diferentes funções: dano em área (AoE), DPS de alvo único e suporte/fortalecimento (buff).
  </Step>
  <Step num="2" title="Maximize as Melhorias Básicas">
    Use o Ouro ganho nas batalhas para melhorar suas unidades ativas. Uma unidade base totalmente melhorada costuma ser mais eficaz do que uma unidade evoluída de nível baixo.
  </Step>
  <Step num="3" title="Colete Materiais de Evolução">
    Colete Cristais de Alma, Super Cristais de Alma e itens específicos de comida, como Carne ou Onigiri. Eles costumam ser concedidos como recompensa de códigos ou ao concluir hordas de nível alto.
  </Step>
  <Step num="4" title="Execute a Evolução">
    Assim que tiver os materiais necessários e a unidade base atingir o nível exigido, use o menu de Evolução para transformá-la.
  </Step>
  <Step num="5" title="Teste de Campo">
    Teste sua unidade evoluída no Modo Infinito (Endless Mode) para ver como o escalonamento dela afeta a velocidade de limpeza das hordas e ajuste sua estratégia de posicionamento de acordo.
  </Step>
</Steps>

<Callout type="success" title="Dica de Evolução">
Apenas evolua unidades que sejam classificadas como Tier A ou Tier S. Evoluir uma unidade de Tier C geralmente é um desperdício de materiais raros, como Fragmentos de Relíquia (Relic Shards) e Super Cristais de Alma, que são melhor guardados para personagens que definem o meta.
</Callout>

</Section>

<Section icon={Target} title="Farm de Recursos e Eficiência">

Gerenciar sua economia é a diferença entre ficar travado na Horda 20 ou alcançar a Horda 50+. O Anime Overseer usa um sistema de várias moedas, onde cada recurso atende a um nicho específico na progressão da conta. Entender onde farmar esses recursos e como gastá-los de forma eficiente é fundamental durante a fase de acesso antecipado.

| Recurso | Fonte Principal | Prioridade de Gasto | Dica para o Jogador |
| :--- | :--- | :--- | :--- |
| **Gemas** | Códigos, Tarefas Diárias | Alta | Guarde apenas para invocar unidades Tier S |
| **Ouro** | Conclusão de Hordas, Batalhas | Alta | Use para subir o nível das unidades imediatamente |
| **Discos de Traço** | Códigos, Desafios | Média | Guarde para unidades que planeja manter a longo prazo |
| **Cristais de Alma** | Hordas de Nível Alto | Guardar | Essencial para o processo de evolução |
| **Fragmentos de Relíquia** | Eventos Especiais | Guardar | Material raro para progressão avançada |

<Section icon={Shield} title="Sistema de Traços e Estratégia de Reroll">

Os Traços fornecem bônus passivos às suas unidades, que variam desde simples aumentos de dano até bônus complexos de utilidade, como custos de posicionamento reduzidos ou maior velocidade de ataque. Como os recursos de rerol são limitados, especialmente no acesso antecipado, você deve ser estratégico sobre quais unidades escolher para otimizar.

**Categorias Comuns de Traços:**
- **Bônus de Dano**: Aumentos diretos no poder bruto.
- **Bônus de Velocidade**: Aumenta a frequência de ataque, excelente para unidades com efeitos de status.
- **Bônus de Alcance**: Permite que as unidades cubram uma parte maior do caminho, essencial para unidades de suporte.
- **Traços de Economia**: Reduz o custo para posicionar ou melhorar unidades durante a batalha.
